# Byte Loading 🚀

Modern FiveM Loadscreen UI  
Designed for **speed, stability, and clean UX**

---

## ✨ Features

- ⚡ Ultra fast loading screen
- 🎨 Clean & modern UI (NUI based)
- 🧠 Manual shutdown support
- 🔄 Smooth progress animations
- 🖱️ Cursor enabled
- 🧩 Easy to customize

---

## 📦 Installation

1. Download / Clone this repository
```bash
git clone https://github.com/bytecorecfx-ui/Byte_loading.git
```
2.Move folder to your resources directory
3.Add to your server.cfg
```bash
ensure Byte_loading
```
